# KiwiHub Website UI Kit

A high-fidelity, interactive click-through prototype of the KiwiHub e-commerce website.

## Screens included
1. **Home** — Hero, category grid, featured products, footer
2. **Shop** — Product grid with filters and search
3. **Product Detail** — Images, specs, add-to-cart, reviews tabs
4. **Cart** — Items list, coupon, order summary, checkout flow
5. **Account** — Login/register, dashboard, orders, addresses, wishlist

## Files
- `index.html` — Main prototype entry; loads all components
- `Navbar.jsx` — Sticky navbar with search, cart badge, mobile menu
- `ProductCard.jsx` — Product card with hover state, badges, add-to-cart
- `Footer.jsx` — Full footer with links, trust badges

## Design Source
- GitHub: `Orange-Mango-2020/kiwi-hub.co.za`
- Fonts: Syne (headings) + Plus Jakarta Sans (body) via Google Fonts
- Colors: See `../../colors_and_type.css`

## Notes
